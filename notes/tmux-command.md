# Tmux command

This note will be a summary of frequently-used tmux command.

## Notes

- vertical split : Ctrl + b and %
- Horizontal split : Ctrl + b and "
- Close active pane : Ctrl + b and x
- Resize pane : Ctrl + b + : and resize-pane [ -D  $SIZE ] [ -U $SIZE]
